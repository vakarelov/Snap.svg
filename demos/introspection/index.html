Snap.svg Introspection Library

About

A modular, reusable library for extracting and displaying class hierarchies, methods, and metadata from Snap.svg and related libraries (eve, mina).

The functionality has been separated into two independent modules located in utils/:

  • utils/snap-introspection.js - Core introspection library (data extraction)
  • utils/snap-introspection-renderer.js - HTML rendering utilities

This separation allows you to use the introspection functionality in any project, with or without the HTML renderer.

Quick Start

Include the libraries in your HTML:

<script src="../../dist/snap.svg.js"></script>
<script src="../../utils/snap-introspection.js"></script>
<script src="../../utils/snap-introspection-renderer.js"></script>

Extract and render:

// Extract data
const data = SnapIntrospection.extractAll({
    Snap: Snap,
    eve: eve,
    mina: mina
});

// Render to page
SnapIntrospectionRenderer.renderToElement('#content', data);

Or just extract data without rendering:

// Just extract - no HTML generation
const snapMethods = SnapIntrospection.extractSnapMethods(Snap);
const classes = SnapIntrospection.extractClasses(Snap);
const json = SnapIntrospection.toJSON(data);
