# Snap.svg Introspection Demos

This folder contains demonstration files for the Snap.svg Introspection library.

## Structure

- **Library Files**: Located in `../../utils/`
  - `utils/snap-introspection.js` - Core introspection library
  - `utils/snap-introspection-renderer.js` - HTML renderer
  - `utils/README.md` - Complete library documentation

- **Demo Files**: Located in this folder (`demos/methods/`)
  - `show_methods.html` - Main demonstration
  - `example-json-export.html` - JSON export example
  - `example-custom-rendering.html` - Custom rendering formats
  - `example-nodejs.js` - Node.js usage examples
  - `index.html` - Navigation and overview page

- **Documentation**: Located in this folder
  - `ARCHITECTURE.md` - Architecture overview
  - `REFACTORING_SUMMARY.md` - Refactoring details
  - `QUICK_REFERENCE.md` - Quick reference card

## Quick Start

1. Open `index.html` to see all available demos
2. Click on any demo link to see it in action
3. View the library documentation in `../../utils/README.md`

## Usage in Your Projects

The utility libraries are now available in the `utils/` folder for use in any project:

```html
<script src="path/to/utils/snap-introspection.js"></script>
<script src="path/to/utils/snap-introspection-renderer.js"></script>
```

See `../../utils/README.md` for complete API documentation and usage examples.
